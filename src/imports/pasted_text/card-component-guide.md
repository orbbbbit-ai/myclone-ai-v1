🧩 CARD COMPONENT STRUCTURE

📐 Card Container
	•	Width: 100% of chat (max 520px)
	•	Height: Auto
	•	Padding: 16px
	•	Border radius: 16px
	•	Background: #FFFFFF
	•	Border: 1px solid #F1F1F1
	•	Shadow:
	•	Y: 4
	•	Blur: 16
	•	Color: rgba(0,0,0,0.04)

Spacing between cards: 12px

⸻

🖼️ TOP: PROJECT PREVIEW IMAGE
	•	Height: 140px
	•	Width: 100%
	•	Border radius: 12px
	•	Object fit: cover

👉 Use:
	•	Clean UI mockup
	•	Not Dribbble nonsense
	•	Real product feel

⸻

🧠 CONTENT SECTION (UNDER IMAGE)

Spacing from image: 12px

⸻

🏷️ PROJECT TITLE
	•	Font: Inter / Medium
	•	Size: 16px
	•	Line height: 22px
	•	Color: #111111

Example:

Fintech Mobile App

⸻

📝 PROJECT DESCRIPTION
	•	Font: Inter / Regular
	•	Size: 14px
	•	Line height: 20px
	•	Color: #666666
	•	Margin-top: 4px

Example:

End-to-end UX design for a scaling fintech startup, focusing on onboarding and transaction flows.

👉 Keep it:
	•	Sharp
	•	Outcome-focused
	•	Not storytelling

⸻

🔘 BOTTOM ACTION ROW

Margin-top: 12px

Layout: Horizontal (space-between)

⸻

🔗 LEFT: CATEGORY TAG
	•	Background: #F5F5F5
	•	Padding: 4px 10px
	•	Border radius: 999px
	•	Font: Inter / Medium
	•	Size: 12px
	•	Color: #444

Examples:
	•	Mobile App
	•	SaaS
	•	Web Platform

⸻

👉 RIGHT: CTA BUTTON

Text:
“View Case Study →”

Style:
	•	Font: Inter / Medium
	•	Size: 13px
	•	Color: #000000

Hover:
	•	Slight underline OR opacity 0.7

👉 Keep it subtle.
Not a loud button. This is premium.

⸻

✨ ADVANCED (PREMIUM TOUCHES)

🧠 1. Hover State (important)

On hover:
	•	Slight lift: translateY(-2px)
	•	Shadow increases slightly
	•	Image zoom: 1.02 scale

👉 Feels alive. Not static.

⸻

🧠 2. “Curated” Signal

Above the cards (AI message):

“Here are a couple relevant projects 👀”

👉 NOT:

“Here are my works”

Language matters.

⸻

🧠 3. Limit Cards

👉 Show max 2–3 cards

Never more.

Scarcity = value.

⸻

💬 HOW IT LOOKS INSIDE CHAT

AI message:

“I’ve got a few solid projects I can show you 👀
Most of my work focuses on product thinking, not just visuals.”

Then:

👇 Cards appear

[Card 1]
[Card 2]

Then:

“What you’re building—does it lean more toward something like this?”

⸻

⚠️ MISTAKES TO AVOID (THIS WILL RUIN IT)
	•	❌ Too many cards
	•	❌ Bright colorful UI (kills premium feel)
	•	❌ Long descriptions
	•	❌ Big loud buttons
	•	❌ Generic mockups

⸻

🔥 FINAL TRUTH

This card is doing 3 jobs:
	1.	Proof → “This guy knows what he’s doing”
	2.	Positioning → “This is not cheap work”
	3.	Filtering → “If you don’t understand this, you’re not my client”
